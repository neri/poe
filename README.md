# MEG-OS Lite

A hobby OS

## Requirements

- IBM PC/AT Compatible / NEC PC-98 Series Computer / Fujitsu FM TOWNS
- Memory 640KB + ?
- Processor ?
- Display
- Standard floppy to boot
- Keyboard and mouse

# Build Environment

* Rust nightly
* nasm
* qemu

### how to build

```
$ make
```

### how to run

```
$ make run
```
